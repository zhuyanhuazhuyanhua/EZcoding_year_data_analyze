# 项目名称

## 项目结构说明

- **`backend`**  
  后端代码，负责与百度 Agent 进行通信。
 
- **`frontend_knowledge_graph`**  
  知识图谱相关的前端代码。

- **`frontend`**  
  前端代码，用于构建用户界面。

- **`static`**  
  静态资源，包括图片、CSS 文件、JavaScript 文件等。

- **`templates`**  
  模板文件，用于前端页面的模板渲染。

- **`lib`**  
  项目依赖的本地库文件。

- **`node_modules`**  
  通过 npm 安装的第三方依赖项。
